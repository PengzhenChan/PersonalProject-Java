# PersonalProject-Java

### 项目描述：阅读《构建之法》并提出问题、使用GitHub提交项目、完成WordCount程序并测试。

### 使用命令行输入

```java
java WordCount 文件名1(例如input.txt) 文件名2(例如output.txt)
```

### 功能简介

读取文件一内容并将以下统计结果写入文件二：

1. 统计字符数
2. 统计单词数
3. 统计有效行数
4. 统计最多的10个单词及其词频

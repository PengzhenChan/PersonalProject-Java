##WordCount
1.功能简介：计算指定文件中的字符数、单词数、非空行数、单词词频，
          并将结果输出在指定文件夹
          
2.使用介绍：命令行运行编译后文件 第一个参数为输入文件位置 第二个参数为输出文件文件位置
如果采用输入文件名，则需要把文件放在编译文件所处的文件夹下，如果输入文件的绝对路径则都可以

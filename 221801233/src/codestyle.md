######                                            CODE STYLE

- 缩进风格:统一缩进4个空格

- 命名风格

  变量采用lowerCamelCase风格,严格java 的驼峰命名方式

  变量不以$开始结束

- 每行最多字符数 100,超过需换行

- 函数最大行数

  70

- 函数,类命名

- 类名:UpperCamelCase风格，必须遵从驼峰形式；

  函数名:lowerCamelCase风格,驼峰命名方式

- 常量命名:

  统一使用大写风格,例如:PAI

- 空行规则

  大括号使用,在上一行,例

  ```java
  public static void main(String args[]){
      
  }
  ```

  函数另起一行

- 注释规则

  使用//或/**/

- 操作符前后空格

  无空格

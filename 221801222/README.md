# PersonalProject-Java（221801222）

## 项目描述

本项目实现英文文本的词频统计，可以统计出总字符数、总单词数，总行数 以及频率top10的单词

## 运行方式

`javac -encoding UTF-8 WordCount.java`

`java WordCount input.txt output.txt`

WordCount(221801330)
==============
项目功能
--------
> 统计文件的字符数
>
> 统计文件的单词总数
>
> 统计文件的有效行数
>
> 统计文件中各单词的出现次数，最终只输出频率最高的10个

运行方式
-----------
> javac -encoding UTF-8 WordCount.java
>
> java WordCount input.txt output.txt
